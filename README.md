# Medical Note Classifier (NLP)

This project demonstrates a simple NLP system that analyzes medical notes and extracts:
- Symptoms
- Conditions
- Medications
- Key phrases

It uses a pretrained biomedical NLP model to perform named entity recognition (NER).  
This is a lightweight academic project inspired by real-world healthcare NLP pipelines.

---

## 🔍 Features
- Medical NER (symptoms, diseases, drugs)
- Keyword-based classification (very simple rule-based tagging)
- Sample medical note included
- Zero training required

---

## 📂 Project Files
- `note_classifier.py` — analyzes the medical note
- `sample_note.txt` — sample clinical note
- `requirements.txt` — dependencies
- `README.md` — full documentation

---

## 🧠 How It Works
1. Load medical note
2. Pass text through a biomedical NER model
3. Extract structured info:
   - Diagnoses  
   - Symptoms  
   - Medications  
4. Apply simple keyword-based classification rules
